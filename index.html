Combinações de Minérios e Usos Industriais

  • Bronze = Cobre + Estanho
  • Latão = Cobre + Zinco
  • Aço = Ferro + Carbono
  • Aço inoxidável = Ferro + Cromo + Níquel
  • Ligas leves = Alumínio + Titânio ou Magnésio
  • Superligas = Níquel + Cobalto + Cromo + Ferro
  • Pigmentos cerâmicos = Cobalto + Zinco + Titânio
  • Ferramentas de corte = Tungstênio + Cobalto
  • Baterias de íon-lítio = Lítio + Cobalto + Níquel
  • Vidros especiais = Sílica + Boro + Alumínio
